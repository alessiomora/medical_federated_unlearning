## Prostate MRI Dataset
ProstateMRI dataset available [here](https://liuquande.github.io/SAML/).

Download the dataset, unzip it if needed. 
In the file `prostate_utility/prostate_utility.py` indicate the location (local path) of the dataset by manually 
changing the content of the variable `base_path`.

## Running Simulations
The file `fed_medical_images.py` contains the minimal code to simulate the federated training.

## Model
The UNet model is customly defined in `prostate_utility/unet.py`.
